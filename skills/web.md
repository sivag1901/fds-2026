## Foundation

This file extends the base FDS 2026 system. All color, typography, and logo rules from `SKILL.md` apply unless explicitly overridden here for the web context.

---

## Colors

**Primary:** Violet — hero backgrounds, CTAs, active states, interactive highlights.
**Accent:** Blue (info), Yellow (attention), Green (success), Orange (alert) — data callouts, status indicators.
**Neutral:** Charcoal and White are the primary surface colors. Black for all headings.
